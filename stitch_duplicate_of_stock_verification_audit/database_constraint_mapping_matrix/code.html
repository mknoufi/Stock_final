Technical Annotations

Optimistic Locking Policy

Frontend assumes success but caches the previous state hash. If Mongo returns a E11000 or version mismatch, the Fork UI resolves the state.

Conflict Resolution Strategy

Manual intervention is required for stock quantity discrepancies between the Oplog and React Native's LocalStore when latency > 5000ms.

Observer Parity

SQL Observer maintains a read-only shadow of all stock movements. Discrepancy checks run every 60s via the global hash comparison.
